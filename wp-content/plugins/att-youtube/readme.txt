Display a YouTube video in your sidebar.

== Description ==

This plugin let you place a YouTube video in your sidebar. Easy to config and there are some options for you.


== Installation ==

1. Upload att-youtube plugin  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select options you want.

== Frequently Asked Questions ==

= How to get YouTube video link? =

Please open the youtube and play a video, press Share button under the video player and copy that link.

= What about option? =

Please read the option and you can select options for your video.
